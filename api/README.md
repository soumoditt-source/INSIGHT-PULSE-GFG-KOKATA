## 🚀 Visionary Features

- **🧠 Gemini 1.5 Pro Core**: Leveraging the world's most advanced reasoning model for data-to-dashboard logic.
- **🧬 Neural Processing Pathway**: Real-time visualization of the AI's internal thought process, schematic reflection, and auto-correction.
- **🛡️ Schematic Reflection**: A self-correcting agentic loop that validates SQL queries against real schema before execution, ensuring 100% accuracy.
- **📊 Multi-Chart Dashboards**: Intelligent intent detection that generates up to 4 synchronized Plotly charts per query.
- **[ENGINE] Zero-Latency Fallback**: Multi-channel API architecture (Google SDK + OpenRouter) with autonomous failover to free models.
- **🛰️ Live Web Integration**: Real-time context injection from DuckDuckGo for news-aware market analysis.

## 🛠️ Performance Architecture

InsightPulse AI follows a strictly scientific **Transformer Reasoning Pathway**:
1. **Atomic Deconstruction**: Breaking user queries into data primitives.
2. **Schematic Mapping**: 1:1 correlation between user intent and database columns.
3. **Recursive Reflection**: AI critiques its own SQL generation for syntax and logic errors.
4. **Visual Optimization**: Selection of optimal chart types (Radar, Sankey, Treemap, etc.) based on data dimensionality.
